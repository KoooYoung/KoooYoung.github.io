Nucleic Acid Converter

This was a biology project that I made out of boredom. It has one feature and that is to convert the user inputted sequences into four different types of nucleic acid, which includes "Sense", "AntiSense", "Codon", "AntiCodon".

This version is the browser port of the original C# version. The C# version has one more exclusive feature and that is to convert the user inputted Codon sequences into different types of protein.

Colorify

Colorify was a part of my assignments for my javascript course. I thought this project could be very useful for web designers to use as it provides you ideas of colors to use for your next website

This website comes with three different features. The first feature is random color generator which you can use to generate all sorts of color. The second feature is RGB slider which is used to generate colors as well. The last feature is cursor RGB, this last feature uses the position of the cursor to generate the three values of RGB colors.

Suit Java

Suit Java was also a part of my assignments for my javascript course. This game is the digitalized version of the traditional Javanese game. It is cleverly named after the programming language that it was created on which is "Javascript".

In this website you will face off against an opponent AI, the first to reach three points win the game.

Spinning Cell

Spinning Cell is a game that I developed using Unity with a couple of friends for an innovation day event. Originally this game was never supposed to be presented in this event, but since I was a bit cocky at the time I pushed it into production so that it'll ready for that event.

The game is about a cell and a virus, in which the objective for the player is to collect as many oxygen orbs that are scattered throughout the level. It combines both rage inducing aspect and educational aspect, which may sound weird for a combination but it turned out to be a very unique experience. The game is available for Windows and Android

Feudal Life

Feudal Life is a text-based life simulator game. It was made purely on console app C# and the direct insipiration for this game is the android game "BitLife", but the era of the game is shifted to the medieval/middle age era. The game however isn't finished yet and still in its development stage.

The player would find itself faced with different kinds of situations, the player would than tasked to choose the best outcome for the character whether its about money, love, or social status.
